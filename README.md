**Employee Attendance Management System (Python & MySQL)**

Introduction
This is a console-based system that helps HR/Admin manage employee attendance, leaves, and monthly reports.

Features
⦁	Add employee details
⦁	Mark attendance (Present/Absent)
⦁	Mark leaves (CL/SL/PL)
⦁	View attendance by date or employee
⦁	Generate monthly attendance reports
⦁	Delete wrong entries

Technologies Used
⦁	Python
⦁	PyMySQL
⦁	MySQL Database
⦁	Datetime Module
⦁	Console/Terminal UI

Future Enhancements
⦁	GUI interface
⦁	Login system
⦁	Export reports as PDF/Excel
⦁	Dashboard summary
