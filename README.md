1. Introduction

This is a console-based application to manage daily employee attendance.
It helps HR/admin easily maintain attendance records, leaves, and monthly reports.

2. Features

Add and manage employee details

Mark attendance (Present/Absent)

Mark leaves (CL / SL / PL)

View attendance by date or employee

Generate monthly reports

Delete wrong entries

3. Technologies Used

Python

PyMySQL

MySQL Database

Datetime Module

Console/Terminal UI

4. Installation Steps

Install Python 3.x

Install required library:

pip install pymysql


Install MySQL and create database

Update database connection in code

Run the project

5. How to Run
python employee_attendance_report_system.py
